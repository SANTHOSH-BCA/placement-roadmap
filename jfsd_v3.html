Part 8
Interview Preparation

Company-specific strategies and must-know question banks for every interview layer.

Service companies
TCS · Infosys · Wipro

Aptitude is the main filter. Coding = arrays/strings easy. Technical = Core Java OOP + DBMS + simple project discussion. Communication is evaluated. Prepare aptitude seriously — that's where most candidates fail.

Product companies
Zoho · Persistent · LTI

Harder coding (medium LeetCode). Spring Boot depth expected. SQL optimization questions appear. Full project walkthrough with cross-questions. Zoho runs multiple coding rounds — raw problem-solving ability matters most.

"Tell me about yourself" — 60-second formula

Opening (10s): "I'm [Name], completing MSc Computer Science at [College]."

Technical identity (20s): "I specialise in Java Full Stack — Spring Boot backend with React.js frontend. Core skills: Spring Boot, REST APIs, MySQL, React."

Project highlight (20s): "I've built [Flagship Project] — full stack with Spring Boot + JWT + React. Live on GitHub and deployed."

Close (10s): "I'm excited about this role because [specific reason]. Ready to contribute from day one."
